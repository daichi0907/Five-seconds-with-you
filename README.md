# どんなゲームか
　死別した弟を救うため、兄（プレイヤー）は5秒後に同じ動作をする弟の幽体と協力してステージを攻略していく「アクションパズルゲーム」です。  
　幽体の弟には物理判定があり、触れることが可能です。そのため、穴の開いた地形に弟を誘導し先へ進むなどといった攻略ギミックなどが出てきます。  
　ゲームの世界観は「絵本の中」となっており、ゲーム中は飛び出す絵本のような形で表現されております。
